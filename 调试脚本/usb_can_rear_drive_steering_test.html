USB-CAN 后驱双驱轮 / 前转向测试

这个页面只做两类真实调试：后驱双驱轮 MSSD 控制器测试，以及前转向 MSSC 控制器测试。现在已按你的接法改成两个独立 COM 口：驱动控制器一个 USB-CAN，转向控制器一个 USB-CAN，互不抢占、互不混线。

1. 双 USB-CAN 连接

驱动侧只接后驱双驱轮 MSSD 控制器，转向侧只接前转向 MSSC 控制器。两个适配器都按 SLCAN 方式通过 Web Serial 连接，默认串口 115200，默认 CAN 速率 S6 / 500 kbps。

2. 后驱双驱轮 MSSD

右轮目标转速寄存器使用 0x39 / 0x3A，左轮目标转速寄存器使用 0x3D / 0x3E。右轮真实转速读取 0x09 / 0x0A，左轮真实转速读取 0x14 / 0x15。本页会把发包、停轮、急停、回读都固定走驱动 USB-CAN。

3. 前转向 MSSC

转向速度命令使用 0x0031，停止使用 0x0030。进入速度闭环前会先写入 0x006B = 0 和 0x006C = 控制模式。位置回读使用 0x0013 / 0x0014，全部固定走转向 USB-CAN。
